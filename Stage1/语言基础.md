## 数组

### 数组的创建方式

#### 动态创建-指定类型与长度即可

int[] a=new int[5];

#### 静态创建--需要指定具体的元素

int[] b=new int[]{1,2,3,4,5};
int[] c={1,2,3,4,5};

### 数组基本特性

1. 我们通过数组的下标来操作数组中的元素,数组的下标从0开始，数组的最大下标为:数组的长度-1
2. 数组的打印char[]底层做了处理，除了它以外，别的类型的数组想要查看具体元素,需要使用数组的工具类Arrays.toString(数组名);
3. 数组的长度,数组的长度指的是数组中元素的个数，数组一旦建立，长度不可改变

### 数组创建过程的分析：如：int[] a={1,2,3,4,5};

1. 在内存中开辟一块连续的内存空间，长度为5
2. 给数组完成初始化的过程，给每个元素赋予对应的默认值，此处是0
3. 数组完成初始化后会分配一个唯一的地址
4. 把数组的地址值会交给引用类型的变量a来保存
5. 根据下标给每个位置上的元素赋值，此处{1,2,3,4,5}

### 数组的遍历

1. 数组的遍历使用的是for循环
2. for循环的循环变量i代表的是数组的下标，最小值是0，最大值是a.length
3. 容易发生ArrayIndexOutOfBoundsException--数组下标越界

### 数组的工具类Arrays

#### toString(数组名)--打印非char类型数组，查看数组具体元素是，需要使用

#### sort(数组名)--直接对传入的乱序数组进行排序

#### copyOf(原数组的名字，新数组的长度)

1. 如果新数组的长度与原数组相同--普通复制
2. 如果新数组的长度小于原数组的长度--数组的缩容
3. 如果新数组的长度大于原数组的长度--数组的扩容
4. TIPS:本方法不会修改原数组，创建的是指定长度的新数组

## 语言结构

### 分支结构

#### 单分支结构

if(判断的条件){
    符合判断条件后要执行的代码，如果不符合条件，跳过此句，执行分支后面的语句
}

#### 多分支结构

if(判断的条件){
    符合判断条件后要执行的代码，如果不符合条件，跳过此句，执行else中语句
}else{
    不满足执行条件，执行代码2
}

#### 嵌套结构

if(要判断的条件1){
    满足此条件，执行语句，否则执行条件2
}else if(要判断的条件2){
    满足此条件，执行语句，否则执行条件2
}else if(要判断的条件3){
    满足此条件，执行语句，否则执行条件3
}else if(要判断的条件4){
    满足此条件，执行语句，否则执行条件4
}else{
    保底选项，如果以上条件都不满足的话，执行该语句，可以不使用
}

### switch结构

1. 结构

   ```java
   switch(a){
       case 1:操作1；break；【可选】
       case 2:操作2；break；【可选】
       case 3:操作3；break；【可选】
       case 4:操作4；break；【可选】
       default：保底操作；【可选】
   }
   ```

2. 括号中的a支持的数据类型：byte short char int String

3. 执行顺序：先拿着变量a的值依次与每一个case作比较，如果相等,就执行case之后的操作，若case后没有break，会发生穿透现象，包括default

4. 如果配置了“保底选项”default，而且没有任何的case被匹配到，则执行default

5. case与default都是可选项，具体怎么加，取决于业务

### for循环结构

#### 概述

循环结构是指在程序中需要反复执行某个功能而设置的一种程序结构。
它由循环体中的条件，判断继续执行某个功能还是退出循环。
根据判断条件,循环结构又可细分为先判断后执行的循环结构和先执行后判断的循环结构。

#### for 形式

```java
for(开始条件；循环条件；更改条件){
        循环体；//满足循环条件后执行的代码
}
```

诀窍：把数据列出来，从哪开始，到哪结束，如何变化
注意事项：++ --自增自减运算符是可以直接改变循环变量本身值的
但是普通的四则运算不可以改变变量的本身的值，所以需要赋值 i=i*10+8（举例）

#### 增强for循环
```java
for(本轮遍历到的元素类型 自定义变量名:要遍历的内容){
    循环体;
}
```

好处:语法简单效率高
坏处:只能从头到尾遍历一次,不能设置步长(普通循环中的更改条件,i如何变化)

### while&do while

```java
while(执行条件){
    循环体；
}
do{
    循环体；
}while(执行条件);
```

while-先判断，再执行
do-while-先执行一次，再判断
如果设置了死循环，必须要设置程序的出口
循环之间是可以互相替换的，但最好使用合适的循环

1. 明确知道循环的次数/需要设置变化情况的时候--使用for循环
2. 需要设置死循环-while(true){}
3. 需要先执行一次，再做判断 do-while

### continue&break

1. continue：跳出本轮循环中continue后面的部分，直接执行下一轮循环
2. break：直接跳出循环，然后执行循环后面的部分
3. 注意：两者之后都不可以写代码，都是不可到达的代码

## 运算符

### 定义

用于连接 表达式 的 操作数，并对操作数执行运算

### 类型

算术运算符、赋值运算符、关系运算符、逻辑运算符、位运算符
按照操作数不同定义：单目运算符、双目运算符和三目运算符

### 自增自减运算符

1. 可以改变变量本身的值
2. 前缀式：先进行变量值变化（+1/-1），再使用
3. 后缀式，先使用，再进行变量值变化（+1/-1）

### 比较运算符（结果是Boolean类型的）

1. ==：比较左右两边的值是否相等，是为true
2. ！=：比较左右两边的值是否不等，不是为true
3. &：和and-->全真才真
4. |：和or-->全假才假
5. &&：双与/短路与，逻辑与单与相同，只不过增加了 “短路”的功能
6. ||：双或/短路或，逻辑与单或相同，只不过增加了“短路”的功能
7. 我们这里说的短路是指：可以提高程序性能，在某些情况下，只需算前半部分就可以得出结果，后半部分就不用算了，也就是被 “短路”了。

### 三目运算符

1？2 : 3	1号位置的表达式判断为true，取2号位置的值，否则取3号位置的值

## 运算规则

1. 计算结果的数据类型，与最大数据类型一致
   int/int→int double/double→double
   double/int→double
2. byte short char三种比int小的类型，运算时会先自动提升成int，再进行运算
3. 整数运算溢出的问题    要避免这个现象，一旦溢出，数据不正确
4. 浮点数会出现运算时不精确的现象--以后解决
5. 浮点数的特殊值 Infinity NAN
